
i. ¿Qué aprendiste sobre el flujo de trabajo con Git y GitHub?
Trabajar con Git y GitHub nos implica gestionar y colaborar en proyectos de desarrollo de software de manera eficiente con nuestros compañeros.
Git es un sistema de control de versiones que te permite seguir los cambios realizados en los archivos, hacer retrocesos 
a versiones anteriores y trabajar con ramas de forma independiente. GitHub es una plataforma basada en la web que hospeda
repositorios Git, facilitando la colaboración entre varios desarrolladores, el seguimiento de problemas y la gestión de proyectos.
La verdad son herramientas muy utiles hoy en dia para poder trabajar mutuamente un proyecto y tener segura nuestra informacion y no estar con el miedo que podamos 
perder la informacion.

iii. ¿Qué ventajas crees que tiene usar Git en proyectos de
programación?
Git nos mejora la organización, tambien facilita el trabajo en equipo y protege nuestro código y te da mucha flexibilidad al gestionar proyectos de programación. No estamos con el temor
